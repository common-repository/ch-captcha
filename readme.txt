CH Captcha use for Add captcha to WordPress on contact form 7.

== Description ==

CH captcha allows to add captcha to WordPress on contact form 7.

You must be installed contact form 7 plugin.

When you activated CH captcha plugin then it will add new tag "CH Captcha" in add/Edit Contact Form page in admin.

You can change following property of captcha noice lavel, backgroud color, font color, noice color etc...

You can use multiple form with captcha code in same page.
   
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `ch-captcha` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

After Activate the plugin you will found new tag (CH Captcha) in contact form add/edit page.
